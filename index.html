Flexbox Row

Use these three properties to create a flexbox row layout.

.row {
display:flex;
flex-direction:row
flex-wrap: wrap;
}

Flexbox Column

Use this to creat a Flexbox column layout.

.column {
display: flex;
flex-direction: column;
}

CSS Grid Layout

Build a 12-column layout using CSS Grid

.grid {
display:grid
width 100%;
grid-template-columns:
repeat(12, lfr);
}

Box Transition Glow

Use transition and box shadows to glow on hover

border-radius: 8px;
transition: all .5s ease-in-out;
}
code-card:hover.

rgba(232, 103, 236, 1), 0 0 15px
