Flexbox Row

Use these three properties to create a flexbox row layout.

.row { display:flex; flex-direction:rowl flex-wrap: wrap; }

Flexbox Column

Use this to creat a Flexbox column layout.

.column { display: flex; flex-direction: column; }

CSS Grid Layout

Build a 12-column layout using CSS Grid

.grid { display:grid width 100% grid-template-columns: repeat(12, lfr); }
